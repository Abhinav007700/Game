Game Instruction of Tiles Game

4x4 Tiles Game

  • There are 16 rounds
  • Clearing each rounds will be awarded with 25 points
  • Total score is 400
  • User advances to the next round if the user selects all the correct tiles.
  • The sequence in which the user selects shouldn't matter.

  • There are 36 rounds
  • Clearing each rounds will be awarded with 25 points
  • Total score is 900
  • User advances to the next round if the user selects all the correct tiles.
  • The sequence in which the user selects shouldn't matter.
